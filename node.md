# Nodes
This file has all the related information about the two different nodes in our project.
## Sensing nodes
These consist of:
1. **ESP32**
2. Battery
3. Moisture sensor
4. Humidity sensor
5. Temperature sensor
6. Light intensity sensor

## Irrigation nodes
These consist of:
1. **ESP32**
2. Electronically controlled water valve

## Data returned by each sensor

1. **Moisture sensor**: This returns a numerical value in the range of *(0-1023)* for the moisture level.
2. **Humidity sensor**: This returns a numerical value in the range of for *()* humidity level.
3. **Temperature sensor**: This returns a numerical value in the range of for *()* temperature.
